Игра Жизнь по заданию на курсе Stepik "Web-фреймворк Flask: введение"
https://stepik.org/lesson/535659/step/6

Тестировалась в браузере Mozilla Firefox Snap на операционной системе Ubuntu

Реализовано:
- основная логика игры;
- автоматизация жизни;
- счётчик поколений;
- отслеживание мёртвых клеток.

Для рецензий создан файл requirements.txt
